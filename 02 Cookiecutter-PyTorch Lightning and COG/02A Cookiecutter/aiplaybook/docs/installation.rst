============
Installation
============


Stable release
--------------

To install lightning-hydra-cookiecutter-template, run this command in your terminal:

.. code-block:: console

    $ pip install aiplaybook

This is the preferred method to install lightning-hydra-cookiecutter-template, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for lightning-hydra-cookiecutter-template can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/aiplaybookin/aiplaybook

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/aiplaybookin/aiplaybook/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/aiplaybookin/aiplaybook
.. _tarball: https://github.com/aiplaybookin/aiplaybook/tarball/master
